# github-mastery

Guide Claude toward professional GitHub collaboration practices.

## Skills

### Drafting Pull Requests

Activates when creating PRs, opening pull requests, or submitting changes for review.

Provides guidance on:
- PR titles using imperative mood with effective verbs
- Description structure: prose-first, headers when needed
- Respecting repository PR templates
- Scope management and self-review

### Writing Issues

Activates when creating issues, filing bugs, or requesting features.

Provides guidance on:
- Issue titles using present-progressive form
- Definition of done with "By the time this Issue is complete..." pattern
- Bug reports with reproduction context
- Feature requests with motivation and scope

## Installation

```bash
claude plugin install github-mastery@notorious-ai
```

## Usage

The skills activate automatically when working on GitHub-related tasks. No explicit invocation needed.
